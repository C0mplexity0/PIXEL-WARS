# PIXEL WARS
Pixel Wars is a simple game on Replit involving pixels and wars.
Wars wars pixels and wars.

## Gamemodes
### Playground
Playground mode is raw pixel wars with no limits on what you can build
- Invincibility
- Infinite Map
- Infinite Coloured Blocks
- No Rules
